### Python Range

Stored in `Range.cpp`

Realization of function like python range. 

We have an ability to use it in this style:

```
for(int x : range(n)){};
for(int x : range(1,n)) {};
for(int x : range(1,n,2)) {};
for(double x : range<double>(n)) {};
```
